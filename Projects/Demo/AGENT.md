# AGENT.md - Demo Project

**Goal**: Learn how DailyAI works
**Tag**: #demo
**Owner**: New Users

## Project Context

This is a demonstration project included with the repository. Use it to understand how tasks flow from daily notes to project folders.

## Workflows

### Adding Tasks
When working on this project:
1. Open your daily note
2. Add tasks with `#demo` tag
3. Example: `- [ ] Read the Demo README #demo`

### Viewing Tasks
1. Check `tasks.md` in this folder
2. Tasks are filed here after running "digest my day"

## Tips

- Use `@Today` to mark critical tasks
- Use `@YYYY-MM-DD` to schedule future tasks
- Tags are removed when tasks are filed (only project tag)
