# Tasks - [Project Name]

Canonical task list for this project. Tasks are automatically filed here from daily notes using the `#project-tag`.

## Task Reference

**Format:** `- [ ] Task description @date #tag`

**Status Markers:**
- `- [ ]` - Not started
- `- [>]` - In progress
- `- [x]` - Completed

**Date Tags:**
- `@Today` - Due today
- `@2026-MM-DD` - Due on specific date
- No date - Undated/backlog

**Example:**
```markdown
- [ ] Complete login screen mockups @2026-02-10 @Sarah #design
- [>] Review API authentication spec @Today
- [x] Setup development environment
```
